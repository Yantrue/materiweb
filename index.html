Pendahuluan: Apa itu GitHub Pages?

GitHub Pages adalah layanan hosting website statis gratis dari GitHub. Cocok untuk portofolio, dokumentasi, atau blog pribadi.

1. Syarat Awal

Siapkan akun GitHub dan repository dengan nama username.github.io.

2. Membuat Repository

Login ke GitHub, klik New Repository, dan beri nama seperti username.github.io.

3. Buat Index.html dan Upload

Buat file index.html di komputer lokal, lalu push ke repository GitHub menggunakan Git atau GitHub Desktop.

4. Aktifkan GitHub Pages

Masuk ke Settings â†’ Pages, pilih branch main, lalu klik Save. Tunggu beberapa saat, web akan aktif di https://username.github.io.

5. Tambahkan CSS & JavaScript

Gunakan folder css dan js untuk menata dan menambahkan interaksi ke websitemu.

6. Kesimpulan

GitHub Pages adalah cara mudah, cepat, dan gratis untuk mempublikasikan situs web statis tanpa hosting berbayar.
